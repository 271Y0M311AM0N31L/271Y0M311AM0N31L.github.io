Proceso de Creación de la pagina web

¿Cómo se diseño la pagina?

La pagina web se diseño a base de HTML y css y un poco de js aplicado por el Bootstrap la pagina se compone de 5 HTML principales, 3 css y 2 js la pagina se diseño usando el concepto multi page el cual combina varias paginas en una, Gracias al Bootstrap studio la pagina tiene diseños personalizados y una interfaz limpia y minimalista

Proceso del Panel Led

¿Cómo Se diseño el panel led?

El panel led se diseño a base de un pedazo de cartón 32X32 el cual se le pego 20 tiras led de 50 cm cada una y estas se unen gracias a 3 cables las cuales van soldados a cada tira led y terminan conectados en un Arduino el cual integra un modulo bluetooth

Proceso de Creación de la Aplicación

¿Cómo se diseño la aplicación?

La aplicación se desarrollo gracias al apoyo de una pagina web y varios tutoriales en YouTube. Esta se hizo en visual estudio code utilizando el leguaje de programación java script esta aplicación utiliza el modulo bluetooth del teléfono para conectarse vía bluetooth al Arduino y darle colores y formas al panel 

Proceso de la base de datos

¿Cómo se diseño la base de datos?

En nuestra base de datos se crearon diversas tablas con los respectivos
departamentos de nuestra empresa que son: Sucursales, empleados, categoría, Clientes, gastos, inventario, productos, ventas y proveedores.                                     
En nuestra base de datos se utilizaron distintas cláusulas dependiendo del uso que se le vaya a dar a estas, por ejemplo “JOIN”, la cual permite combinar registros de una o más tablas en una base de datos, o “SELECT”, que se usa para especificar los nombres de los campos que contienen los datos que quiere usar en una consulta.
